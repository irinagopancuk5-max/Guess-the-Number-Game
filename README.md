# Guess the Number Game

A simple C++ game where the computer generates a random number and the player tries to guess it.

---

## ⚙ Features
- Random number generation
- User input
- Hints (too high / too low)
- Loop until correct answer

---

## 🎯 What I learned
- Loops (while)
- Random numbers
- Game logic
- Conditions
