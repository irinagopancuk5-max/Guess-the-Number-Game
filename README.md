# Guess the Number Game
## 📌 About
A simple game where the computer randomly selects a number and the player tries to guess it.

## ⚙ Features
- Random number generation
- User input
- Hints (too big / too small)
- Loop until correct answer

## 🎯 What I learned
- Loops (while)
- Random numbers
- Conditions
- Game logic
